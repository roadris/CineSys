CineSys

Descrição:

No início do 1º Semestre de 2014 o Prof. Edy Segura propôs um trabalho a ser entregue no final desse mesmo semestre. Esse trabalho deveria atender algumas exigências como: Um CRUD (Cadastrar - Recuperar - Atualizar - Deletar) dos dados, geração de relatórios, em síntese um projeto que ajudasse no gerenciamento. Pensando nisso resolvemos desenvolver um projeto para auxiliar no gerenciamento de cinemas, onde desenvolvemos uma parte para facilitar o acesso do cliente em relação a saber qual a programação das sessões e quais serão futuros filmes a serem exibidos e outra parte para o gerenciamento, onde os funcionários podem fazer lançamentos de compras e vendas e verificar alguns relatórios. Como também foi proposto pelo Prof. Márcio Azevedo a apresentação de um trabalho na disciplina de Engenharia de Software III, utilizamos o mesmo projeto do cinema para identificar requisitos, casos de usos, fluxo de eventos, diagramas de comunicação, casos de teste e UCP (Use Case Points).

Objetivo:

Facilitar o gerenciamento do cinema.

Características

Todo sistema pode ser configurado com cores e logomarca de acordo com a sua vontade.
